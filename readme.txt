

Este proyecto es una maqueta web básica de una cafetería llamada Café Venecia


La página contiene los siguientes elementos principales:

Encabezado (Header):
   - Logo del negocio.
   - Menú de navegación con anclas a secciones de la misma página.

Sección de Bienvenida 
   - Imagen de fondo atractiva.
   - Mensaje de bienvenida con el eslogan del café.

Menú Destacado:
   - Presentación de algunos productos (Café Latte, Capuccino, Brownie) con sus respectivos precios.


Pie de página (Footer):
   - Redes Sociales , Direccion y Telefonos
